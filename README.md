# Power BI Case Study

## Project Overview
This project demonstrates the development of an interactive dashboard using Power BI to analyze business data and generate actionable insights. The dashboard helps track key performance indicators (KPIs), identify trends, and support data-driven decision making.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key Features
- Interactive dashboard with filters and slicers
- KPI tracking and performance analysis
- Trend and category-based insights
- Data cleaning and transformation using Power Query
